Особливості передачі конфіденційних даних 
під час виконання програм через GitHub Actions

Якщо програмам необхідно передати конфіденційні дані,
наприклад, ім'я та пароль користувача, 
необхідно ці значення зберігати у спеціальних секретних змінних.
Для створення секретних GitHub-змінних 
необхідно:
1) зайти до будь-якого репозиторія, яким ви є власником
2) перейти до сторінки за шляхом "Settings/Secrets/Actions"
3) створити змінні з необхідними секретними значеннями:
LOGIN
PASSWORD

В подальшому на значення цих змінних можна посилатися:
${{ secrets.LOGIN }} 
${{ secrets.PASSWORD }}
